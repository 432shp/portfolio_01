Director's Philosophy

私のディレクター像

WEBディレクターとは、「誰のために、何を、なぜ作るか」を定義する職種だと考えています。 デザインやコーディングの善し悪しを判断しながら、クライアントのビジネス課題とユーザーの体験を橋渡しする——そのハブとなる役割に魅力を感じ、この道を志しました。

今の私に足りないのは実務経験です。しかし、企画・設計・進行管理の思考プロセスは、 このポートフォリオを通じてお見せできると確信しています。 まずは1年間、現場の最前線で修行をさせていただきたいと考えております。

VIGILO — 制作ドキュメント

High-End Gym & Sauna Site

VIGILOは、高級フィットネス施設を想定したフルスクラッチ制作のコンセプトサイトです。 「覚醒・静寂・圧倒的な肉体美」というコンセプトを軸に、 ターゲット設定 → 競合調査 → サイト構成 → 制作 → 評価 というディレクタープロセス全体を一人で担いました。

PROJECT INFO
役割ディレクション / コーディング
期間約3週間
ページ数5P（Home / About / Facility / Sauna / Contact）
技術HTML / CSS / JavaScript

制作プロセス

STEP 01 目的定義

誰に・何を・なぜ伝えるかを言語化。Membershipへの誘導をゴールに設定。

STEP 02 競合調査

高級ジム・スパ系サイト5社を比較。差別化ポイントとして「静」の世界観を決定。

STEP 03 IA・WF設計

サイトマップ→ワイヤーフレームの順に設計。ページ間の導線を整理。

STEP 04 制作・実装

デザイン→コーディングを一人で担当。ローディング・スライダーも実装。

STEP 05 振り返り

良かった点・改善点をドキュメント化。次回への学びを明文化。

04

振り返りと学び

Reflection

できたこと

コンセプト「覚醒・静寂」を世界観・コピー・デザインで一貫して表現できた。ターゲット設定からコピーライティングまで、企画レベルで軸がぶれなかった。

5ページ構成で導線を意識したIA設計を実施。Home → 各カテゴリ → Contact という流れを作ることができた。

ローディングアニメーション・画像スライダーなど、演出面でユーザー体験を意識した実装ができた。

次の課題

CVボタンの設置が少なく、コンバージョン導線が弱い。ディレクターとして「目標数値から逆算した設計」をする必要があると実感した。

ワイヤーフレームをFigmaなどのツールで正式に作成し、設計の意図を可視化・説明できる形にする必要がある。

スマートフォン表示のレスポンシブ対応が不完全。今後はモバイルファーストで設計する意識を持ちたい。
